# Training Planner (План тренировок)

**Автор:** Попов Роман

## Описание
Это приложение для планирования и учета тренировок. Можно добавлять записи, фильтровать по типу и дате, сохранять данные в JSON и загружать их при старте.

## Инструкции по запуску
1. Убедитесь, что установлен Python 3.
2. Скачайте папку с проектом или клонируйте репозиторий.
3. Запустите файл `training_planner.py` командой:
```bash
python training_planner.py
